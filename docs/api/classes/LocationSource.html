    Class LocationSourceAbstract

    Abstract base class for all location data providers.

    Subclass this to implement a new location source. Implement start() and stop(), then call the protected emitPosition, emitStatus, and emitError helpers when data is available.

    Example

    class MyGpsSource extends LocationSource {
      readonly sourceId = 'my-gps';
      start(): void { // begin receiving data }
      stop(): void  { // stop and release resources }
    }
